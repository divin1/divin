## divin.me

Source code for my website/blog @ [divin.me](https://divin.me).

Built with:

* Next.js 15
* React 18
* Tailwind CSS
* Headless UI
* Motion
* Lucide icons

Deployed via Vercel.

## Getting Started

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
